# DeepSeek Bot Manager

A comprehensive web-based application for managing AI bots powered exclusively by the DeepSeek V3.2 API. This application provides a modern, intuitive interface for creating, managing, and interacting with AI bots, complete with analytics, chat history, and GitHub integration.

## ✨ Features

### 🤖 Bot Management
- **Create & Edit Bots**: Customize bots with names, system prompts, temperature, and token limits
- **Bot Status**: Visual indicators for active/inactive bots
- **Search & Filter**: Quickly find bots by name
- **Provider Badges**: Clear identification of DeepSeek-powered bots
- **Message Count Tracking**: See conversation history at a glance

### 💬 Intelligent Chat Interface
- **Real-time Conversations**: Chat with selected bots
- **Message History**: Persistent chat storage with timestamps
- **Typing Indicators**: Visual feedback during AI processing
- **Token Usage Display**: Real-time token consumption tracking
- **Export Functionality**: Save conversations as text files
- **Auto-scroll**: Always see the latest messages

### 📊 Analytics Dashboard
- **Usage Statistics**: Total bots, active bots, messages, and tokens
- **Visual Charts**: Line charts for message trends, doughnut charts for token distribution
- **Activity Feed**: Recent bot interactions and events
- **Performance Metrics**: Response times and API usage

### ⚙️ Settings & Configuration
- **API Key Management**: Secure storage for DeepSeek API keys
- **Model Settings**: Adjust temperature and max tokens globally
- **Theme Toggle**: Light/dark mode with persistent preferences
- **Data Management**: Export/import bot configurations
- **GitHub Integration**: Push your bot configurations to GitHub repositories

### 🔒 Security & Reliability
- **Local Storage**: All data stored locally on your server
- **API Error Handling**: Graceful degradation and retry mechanisms
- **Input Validation**: Frontend and backend validation
- **Rate Limit Awareness**: Visual warnings for API limits
